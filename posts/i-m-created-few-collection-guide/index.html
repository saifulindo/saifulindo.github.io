Saya membuat ini dimulai dari berbagi, kebutuhan, komunitas dan pekerjaan. Sebenarnya semua dimulai dari berbagi, saya menuliskan inipun juga berbagi.

“Belajar itu menyenangkan, bisa berbagi akan lebih menyenangkan”. Panduan itu menjadi penting apabila memang dibutuhkan, saya kira setiap manusia membutuhkan atau bahkan memiliki panduan, karena dengan panduan sesuatu yang tidak diketahui menjadi diketahui.

Sekarang yang menjadi permasalahan adalah bagaimana membuat diri kita butuh akan ilmu, bagaimana akan diberi pemahaman kalau dengan ilmu saja tidak membutuhkan, maka siap-siaplah menahan perihnya kebodohan kalau dengan ilmu saja tidak menghiraukan. Al-imam As-Syafi’i mengatakan “Jika kamu tidak sanggup menahan lelahnya belajar, maka kamu harus sanggup menahan perihnya kebodohan”.

Adab itu didahulukan diatas ilmu, jangan meremehkan doa, yakinlah bahwa Allah-lah yang memiliki ilmu, ilmu Allah meliputi langit dan bumi dan Allah-lah yang menjadikan kita dapat memahami ilmu serta butuh akan ilmu, maka keduanya itu semua dibutuhkan doa.

Berikut ini akan saya ulas setiap panduan, yaitu ikhtisar secara umum mengenai isi panduan serta sebagai penambah informasi bagi anda :

Install dan konfigurasi Git

Panduan ini dibuat ketika mengadakan workshop Git yang diadakan oleh Komunitas CAH UNP, ada dua panduan yaitu untuk pengguna windows dan linux. Sekilas tentang Git, Git merupakan sistem pengontrol versi yang terdistribusi (distribusion version control system), yang digunakan untuk mengontrol versi sebuah atau beberapa berkas. Git lebih banyak dimanfaatkan oleh para Aplication Development, sebenarnya Git juga dapat di gunakan oleh orang perkantoran untuk memanajemen berkas perkantoran mereka, hanya saja mereka orang kantoran tidak terbiasa menggunakan version control, maka version control lebih banyak digunakan oleh programmer.

Contekan Git

Baiklah dengan dua panduan sebelumnya saya kira sudah dapat mencoba untuk ikut berkontribusi pengembangan contekan-git di GitHub, Contekan Git ini maksudnya yaitu berupa selembaran perintah – perintah mendasar tentang pengoperasian penggunaan Git, meskipun tidak enak didengarkan, itu saya ambil dari kata Cheatsheet.

Cara menggandakan contekan-git menggunakan Git :

$ mkdir GitHub
$ cd ~/Github/
$ git clone https://github.com/saifulindo/contekan-git.git
Cloning into 'contekan-git'...
....
.........
Checking connectivity... done.
# cd contekan-git

File bernama contekan-git.docxbisa dibuka, dan bisa langsung di print untuk dipelajari.

Git Sebagai Media Pembelajaran

Maksud dari Pembelajaran tersebut, terbatas pada pelajaran atau mata kuliah pemrograman, pada panduan ini Git Server-nya menggunakan GitHub sebagai social coding, Bapak/Ibu Guru atau Dosen bisa melakukan request private repository ke education.github.com, mungkin ada beberapa proyek anak-anak yang tidak menginginkan di publikasi.

Sebenarnya menerapkan metode Open Source terhadap beberapa proyek yang kita kerjakan lebih mempercepat proses berfikir dan menghadapi tantangan percepatan tersebut. Jadi proyek anak – anak yang sudah release bisa di opensource-kan untuk perkembangan lebih lanjut. Dinyatakan bahwa proyek-proyek yang dilacak di Black Duck Knowledgebase, mencerminkan banyak, yayasan dan organisasi yang membentuk komunitas open source secara keseluruhan.
